# Q013 Passing

## 解法
AGC王国を1つのグラフと見たとき, 単に「街xから他の街への最短経路を求めよ」という問題であれば, Dijkstra法など有名アルゴリズムが存在することが知られています.

今回は, 「街1→街k→街N」という経路が指定されていますが, これを「街1→街k」と「街k→街N」の部分に分けて考えます. それぞれの値はDijkstra法で計算できるので, その合計である本問の答えも計算できるようになります.
