# 054 Takahashi Number

## 解法
著者と論文の両方をノードとしたグラフを構築し, 最短経路問題を適用すると解くことができる問題です.
いわゆる「解説AC」なので, 詳細な内容については[公式解説](https://github.com/E869120/kyopro_educational_90/blob/main/editorial/054.jpg)を参照してください.
